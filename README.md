# stardew-gifts-web

- setup using `vite`: https://vitejs.dev/guide/

- to build the app
  ```
  npm run build
  ```
  
- to start the development server:
  ```
  npm run dev
  ```

## Libraries used
- Material UI
  - to install MUI: `npm install @mui/material @emotion/react @emotion/styled @mui/icons-material`
